# Changelog


## [1.1.0] - 2025-11-29

### Добавлено

- новый вид генерации nfc-generation
  более редкий спавн руды по канонам NFC мода для беты 1.7.3

## [1.0.0] - 2025-11-29

### Добавлено

- Базовая функциональность плагина для настройки частоты спавна руды
- Поддержка настройки множителей для всех основных руд:
  - Угольная руда (coal_ore, deepslate_coal_ore)
  - Медная руда (copper_ore, deepslate_copper_ore)
  - Железная руда (iron_ore, deepslate_iron_ore)
  - Золотая руда (gold_ore, deepslate_gold_ore)
  - Алмазная руда (diamond_ore, deepslate_diamond_ore)
  - Редстоун руда (redstone_ore, deepslate_redstone_ore)
  - Лазурит руда (lapis_ore, deepslate_lapis_ore)
  - Изумрудная руда (emerald_ore, deepslate_emerald_ore)
  - Блоки сырого железа (raw_iron_block)
  - Блоки сырой меди (raw_copper_block)
  - Адские руды (nether_gold_ore, nether_quartz_ore, ancient_debris)
- Команда `/orecontrol` (алиас `/oc`) для перезагрузки конфигурации
- Конфигурационный файл с настройками для каждого мира
- Кастомный генератор руды с поддержкой множителей
- Удаление стандартной генерации руды при использовании кастомных настроек

### Исправлено

- Исправлена проблема с бесконечной рекурсией при генерации руды
- Оптимизирована работа с чанками для предотвращения зависаний сервера
- Отключено обновление физики при замене блоков для повышения производительности

### Технические детали

- Использование Java 21
- Совместимость с Paper/Spigot API 1.21.1
- Maven для сборки проекта
