# OreControl

Плагин для Minecraft сервера (Paper 1.21.10), позволяющий настраивать частоту спавна всех видов руды в разных мирах.

## Возможности

- Настройка частоты спавна всех видов руды через конфигурационный файл
- Отдельные настройки для каждого мира (world, world_nether и т.д.)
- Поддержка всех видов руд:
  - Обычные руды: уголь, медь, железо, золото, алмазы, редстоун, лазурит, изумруды
  - Глубинные версии всех руд (deepslate)
  - Адские руды: кварц, золотая адская руда, незерит

## Установка

1. Соберите плагин с помощью Maven:
```bash
mvn clean package
```

2. Скопируйте файл `target/OreControl-1.0.0.jar` в папку `plugins` вашего сервера

3. Перезапустите сервер

4. Настройте файл `plugins/OreControl/config.yml` под свои нужды

## Настройка

Откройте файл `plugins/OreControl/config.yml` и настройте множители для каждой руды.

### Множители:
- `1.0` = нормальная частота (по умолчанию)
- `0.5` = в 2 раза меньше руды
- `2.0` = в 2 раза больше руды
- `0.0` = отключить спавн этой руды

### Пример конфигурации:

```yaml
worlds:
  world:
    diamond_ore: 0.5  # Алмазов в 2 раза меньше
    deepslate_diamond_ore: 0.5
    iron_ore: 2.0  # Железа в 2 раза больше
    deepslate_iron_ore: 2.0
    
  world_nether:
    nether_quartz_ore: 3.0  # Кварца в 3 раза больше
    ancient_debris: 0.1  # Незерита в 10 раз меньше
```

## Команды

- `/orecontrol` или `/oc` - показывает информацию о плагине
- `/orecontrol reload` или `/oc reload` - перезагружает конфигурацию без перезапуска сервера

**Права доступа:**
- `orecontrol.use` - базовое право на использование команды (по умолчанию: op)
- `orecontrol.reload` - право на перезагрузку конфигурации (по умолчанию: op)

## Важные замечания

- Плагин работает только с **новыми чанками**, которые генерируются после установки плагина
- Для применения изменений к существующим чанкам необходимо их регенерировать или удалить регион
- После изменения конфигурации используйте команду `/orecontrol reload` для применения изменений без перезапуска сервера

## Компиляция

Требования:
- Java 21
- Maven 3.6+

Команда для сборки:
```bash
mvn clean package
```

Готовый JAR файл будет находиться в `target/OreControl-1.0.0.jar`
